# Research

This directory stores research reports and investigation outputs produced during project development.

## Naming Convention

```
Topic_Research_YYYYMMDD.md
```

- **Topic**: CamelCase subject (e.g., `Codex_Claude_Code_MultiAgent`)
- **YYYYMMDD**: Date the research was conducted

## Contents

Research artifacts only — not skills, agents, or project source code. Typical contents include:

- Ecosystem surveys and competitive analysis
- Technology evaluation reports
- Best practice investigations
- Decision records backed by external research
