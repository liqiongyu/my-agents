## [0.1.2] - 2026-03-30

### Changed
- Expanded the skill description to cover agent orchestration plus agent runtime or platform design more explicitly.
- Added a concrete `example-artifact-pack.md` reference and linked it from the core workflow and methodology reference.
- Expanded the trigger-posture eval suite with ambiguous near-miss cases for broad AI features, MCP-only questions, and chatbot-with-tools requests.
- Clarified that the framework comparison reference is intentionally non-exhaustive and should hand off to `deep-research` when a consequential option is missing.

## [0.1.1] - 2026-03-30

### Changed
- Excluded `CHANGELOG.md` from runtime projections to keep installed surfaces leaner.
- Clarified the boundary between agent-system design and narrower execution-focused agentic coding workflows.
- Replaced repo-specific validation commands in `SKILL.md` with a runtime-safe contributor note.
- Added freshness metadata and recheck guidance to the framework selection reference.

### Added

- Reference guides for methodology, system patterns, framework selection, and evaluation/operations.
